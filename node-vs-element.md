# 节点和元素的区别

- 节点
  - DOM文档内的所有内容
  - 包括text，注释

- 元素
  - 只有html标签
  - 是节点的一种，元素节点

## method 的区别

> **element的方法**
>
> - getelementby...(返回htmllist/element)
> - chiledren和parentelement
> - elementsibling
>
> **node的方法**
>
> - parentnode和childnodes
> - queryselector
> - siblings

**queryselectorall和getelementbyclassname的区别：前者返回静态的nodelist，后者返回随时更新的htmllist，推荐前者**
